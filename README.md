# MakerHub Documentation

Control your CNC machine with any device, from anywhere.

## Quickstart

- [Install](installation) one or more [hubs](interface/hubs).
- [Connect](machines/connections/new.md) to your first machine.
- [Calibrate](machines/calibration) your machine (if required).
- [Run](interface/tools/programs.md) your first [gcode program](machines/programs).
